How to do a pull request on github

Steps to doing a pull request on github

  1. Navigate to the repository in question's main page
  2. In the branch menu, find the branch containing your commits
  3. Find the 'compare and pull request' button and click it to start a pull request for the selected branch
  4. Use the base branch menu on the left to select the branch you'd like to merge your changes into. Use the menu next to it on the right to choose the branch your changes were made in
  5. Type a title and desription for your pull request
  6. Finally, click 'create pull request'

Additional tips for doing a pull request

  • Pull requests can only be opened between two branches
